# phi-2 Code Generation Web

This project is a scaffold for a local phi-2 code generation web app built with LangChain and Gradio.

## Structure

- `app.py` - entrypoint
- `src/` - core modules
- `tests/` - test files
- `docs/` - architecture and notes
- `scripts/` - helper scripts

## Status

The repository currently contains the planned project structure and placeholder files.
